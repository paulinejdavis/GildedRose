# Gilded Rose
This is the Gilded Rose kata in JavaScript with Jest

## Getting started

Install dependencies

```
npm install
```
Running tests

## To run all tests

```
npm test
```
To run all tests in watch mode

```
npm run test:watch
```
To generate test coverage report

```
npm run test:coverage
```
